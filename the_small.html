In Short

Why use Oxygen XML Author?

Oxygen XML Author, is a tool designed for technical writing as well as other purposes. It is mostly recommended in the field of technical writing and this inspired me to learn the application, and see what all the fuss is about.

I managed to create my first map, and produce an output of HTML & PDF format for the same map.

About Oxygen XML Author:
  • Free trail period, valid for 30 days
  • Unless you have previous experience with similar applications, a quick course might be recommended
  • It should be noted that there is also other applications available with Oxygen, including Developer, Editor and Web Author
  • Depending on the desired goal for using the application, you might want to review additional applications as well

Things to note when using the application

  1. Language used with this new learning experience: Darwin Information Typing Architectuere, DITA.
  2. There was some security restrictions that needed to be changed in order to start creating maps, topics, concepts etc.
  3. The overall use is fearly easy to understand, yet the structured document might become more complex, depending on the desired outcome.
  4. Overall process to transform your informaion to a desired outut, is quite straight forward.
  5. Output paramaters for both PDF document and HTML link, was both neatly structured final deliverables.
